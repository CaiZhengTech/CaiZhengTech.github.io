TicketTrading

CS 320 - Final Project

I built a full-stack ticket trading platform with my classmates, using the MERN stack that allows users to buy and resell event tickets through a limit order system. Users can create accounts, manage their balance, and place bids or offers for specific events. The app maintains an order book that matches bids and offers based on price and quantity, facilitating peer-to-peer transactions. I implemented features like account balance updates, transaction confirmations, and ticket ownership transfers, all while simulating real-world trading mechanics within a secure, user-friendly interface.

SentiPredict

Sentiment Analysis Financial Tool

As part of my research at ALT Labs, I developed a comprehensive NLP pipeline to analyze sentiment in financial news and explore its relationship with market trends. I used RoBERTa for fine-grained sentiment scoring and BART for summarizing lengthy articles, allowing me to separately evaluate headline and content sentiment. My goal was to determine how effectively sentiment analysis could predict stock movements compared to traditional market indicators.

Dungeon Quest

HackUMass XII Game

I developed a pixel art dungeon crawler game inspired by classics like Zelda and Diablo, built entirely for the browser using React, Phaser.js, Vite, and Howler.js. The game features real time combat, level navigation, custom audio effects, and a responsive UI. I implemented core mechanics such as collision detection, player enemy interactions, health management, and a modular architecture for expanding content. Through this project, I deepened my understanding of game engines, canvas rendering, JavaScript modular design, and cross-browser audio integration. I'm currently working on adding collectibles, boss fights, and a backend-based save system using Node.js, Express, and MongoDB.
